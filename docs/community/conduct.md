# Code of Conduct

The Lind project follows the [Contributor Covenant][homepage] Code of Conduct.

The canonical, authoritative version lives at the root of the repository:
[**CODE_OF_CONDUCT.md**](https://github.com/Lind-Project/lind-wasm/blob/main/CODE_OF_CONDUCT.md).
Please refer to it for the full text, including our standards, scope, and
enforcement guidelines.

To report unacceptable behavior, contact the community leaders responsible for
enforcement at
[lind-project-reporting@googlegroups.com](mailto:lind-project-reporting@googlegroups.com).
All reports are handled promptly, fairly, and confidentially.

[homepage]: https://www.contributor-covenant.org
